7Zip

Бесплатный и хороший архиватор! Открывает архивы как Wim, Zip, 7z, tar.gz и многие другие.

Для Windows XP/7/8/8.1/10/11

VLC

Бесплатный и свободный кросс-платформенный медиаплеер и медиаплатформа с открытым исходным кодом.

Для Windows 7/8/8.1/10/11
Также он доступен для Linux
Дистрибутивы: Ubuntu, Debian, Arch и другие!

Arch: #pacman -S vlc
Debian/Ubuntu: $sudo apt install vlc
snap: $sudo snap install vlc

qBittorrent

Отличный торрент трекер, минималистичен, отсуствие рекламы, открытый исходный код!

Дистрибутивы: Ubuntu, Debian, Arch и другие!

Notepad++

Свободный текстовый редактор с открытым исходным кодом для Windows с подсветкой синтаксиса, разметки, а также языков описания аппаратуры VHDL и Verilog.
Для Windows XP/7/8/8.1/10/11

CCleaner

Условно-бесплатная утилита с закрытым исходным кодом, которая предоставляет пользователям простой в использовании инструмент для очистки и оптимизации 32-разрядных и 64-разрядных операционных систем Windows. Утилита была создана британской частной фирмой Piriform Limited и написана на C++

Firefox

Свободный браузер на движке Quantum, разработкой и распространением которого занимается Mozilla Corporation. Четвёртый по популярности браузер в мире и первый среди свободного ПО - в марте 2021 года его рыночная доля составила ▼7,49%. В России Firefox занимает четвёртое место по популярности среди браузеров для ПК с долей 5,26% пользователей на март 2021 года.

LibreOffice

Кроссплатформенный, свободно распространяемый офисный пакет с открытым исходным кодом, созданный как ответвление OpenOffice.org в 2010 году. Разрабатывается сообществом из более чем 480 программистов под эгидой некоммерческого фонда The Document Foundation за счёт пожертвований отдельных лиц и организаций.

Thunderbird

Thunderbird - это бесплатный кроссплатформенный клиент с открытым исходным кодом для электронной почты, новостей и обмена мгновенными сообщениями. Стратегия проекта моделируется после Mozilla Firefox.

XnView

Кроссплатформенная программа для просмотра изображений, поддерживающая просмотр более 400 и сохранение до 50 различных графических и мультимедийных форматов файлов. Программа распространяется бесплатно для некоммерческого и образовательного использования, однако существовала и платная версия. XnView по функциональным возможностям не уступает таким известным и популярным средствам просмотра графических форматов, как программы ACDSee и IrfanView. XnView работает со сканером и принтером, а также в полнокомплектной версии умеет проигрывать широко распространённые аудио-видео форматы файлов, но только при условии наличия нужных кодеков в системе.
